django-gishelper
#################

**Gishelper** is python django application with useful commands for django.gis module.

.. contents::

Requirements
-------------

- python >= 2.5
- pip >= 0.8


Installation
------------

**Gishelper** should be installed using pip: ::

    pip install git+git://github.com/klen/django-gishelper.git
